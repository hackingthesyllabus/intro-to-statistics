Quick recognition guide

📊

Numeric outcome

mean · average · score · years · amount

→ Use a t procedure

📋

Categorical outcome

proportion · percent · yes/no · rate

→ Use a z procedure

🔲

Two categorical variables

association · relationship · table of counts

→ Use chi-square

📏

Estimating, not testing

construct · estimate · how confident

→ Use a confidence interval

↔️

Two-tailed test

different from · not equal to

→ HA: parameter ≠ value

→

One-tailed test

greater than · less than · more likely · higher

→ HA: parameter > or < value

Before you pick a formula

1
Are you estimating or testing?
Estimating → confidence interval. Testing → hypothesis test.
2
What kind of outcome variable?
Numeric → t test. Categorical → z test. Both categorical → chi-square.
3
One sample or two?
Comparing to a known value → one sample. Comparing two groups → two sample.
4
One-tailed or two-tailed?
"Different" → two-tailed. "Greater/less than" → one-tailed.

Confidence intervals

CI

Confidence interval for a mean

Use when: numeric outcome, estimating a population mean, unknown population σ.

Ȳ ± t* (sY / √N)
Ȳsample mean
t*critical t value (from t table, df = N − 1)
sYsample standard deviation
Nsample size
Example cue

"Construct a 95% confidence interval for the average burnout score."

Interpretation

"We are 95% confident that the true population mean lies between [lower] and [upper]."

CI

Confidence interval for a proportion

Use when: categorical outcome, estimating a population proportion.

p̂ ± z* √(p̂(1 − p̂) / n)
p̂sample proportion
z*critical z value (1.96 for 95%, 2.58 for 99%)
nsample size
Example cue

"Estimate the proportion of adults who support the policy and give a 95% confidence interval."

Interpretation

"We are 95% confident that the true population proportion lies between [lower] and [upper]."

t tests — numeric outcome

t

One-sample t test

Use when: numeric outcome, one sample mean compared to a known or hypothesized population mean.

t = (Ȳ − μ0) / (sY / √N)
Ȳsample mean
μ0hypothesized population mean (from H₀)
sY / √Nstandard error of the mean
dfN − 1
Example cue

"Is the average number of years of education significantly different from the national average of 12.2 years?"

Five-step model
  1. State H₀ and HA (e.g., H₀: μ = 12.2; HA: μ ≠ 12.2)
  2. Choose α (usually 0.05) and identify df = N − 1
  3. Find tcritical from the t table
  4. Calculate tobtained using the formula
  5. Compare: if |tobtained| > tcritical, reject H₀
Interpretation

"There is [not] a statistically significant difference between the sample mean and the hypothesized population mean (t = [value], df = [N−1], p [</≥] 0.05)."

t

Two-sample t test

Use when: numeric outcome, comparing the means of two independent groups.

t = (Ȳ1 − Ȳ2) / SEȲ₁−Ȳ₂
SEȲ₁−Ȳ₂ = √( s1² / N1 + s2² / N2 )
Ȳ1, Ȳ2means of the two groups
s1, s2standard deviations of each group
N1, N2sample sizes of each group
df(N1 − 1) + (N2 − 1)
Example cue

"Is there a gender difference in job satisfaction?" (satisfaction measured on a numeric scale)

Five-step model
  1. H₀: μ1 = μ2; HA: μ1 ≠ μ2 (or > / < for one-tailed)
  2. Choose α; calculate df = (N1−1) + (N2−1)
  3. Find tcritical from the t table
  4. Calculate SE and then tobtained
  5. Compare: reject H₀ if |tobtained| > tcritical
Interpretation

"There is [not] a statistically significant difference in [outcome] between [group 1] and [group 2]."

Tail check: "Different from" or "not equal to" → two-tailed test (divide α by 2, use ±tcritical). "Greater than" or "less than" → one-tailed test (use only one side of the distribution).

z tests — categorical outcome (proportions)

z

One-proportion z test

Use when: categorical outcome, one sample proportion compared to a known or hypothesized population proportion.

z = (p̂ − π0) / sπ₀
sπ₀ = √( π0(1 − π0) / n )
p̂sample proportion
π0hypothesized population proportion (from H₀)
sπ₀standard error of the proportion under H₀
nsample size
Example cue

"Is the poverty rate less than 5%?" / "Are older people more likely to be victimized?" (compared to a known rate)

Five-step model
  1. H₀: p = π0; HA: p < π0 (or > or ≠)
  2. Choose α; find zcritical from z table (±1.96 for two-tailed α=0.05)
  3. Calculate sπ₀
  4. Calculate zobtained
  5. Compare to zcritical; reject H₀ if |zobtained| > zcritical
Interpretation

"There is [not] sufficient evidence to conclude that the population proportion is [greater than / less than / different from] π0."

z

Two-proportion z test

Use when: categorical outcome, comparing proportions for two independent groups.

z = (p̂1 − p̂2) / SEp̂₁−p̂₂
SEp̂₁−p̂₂ = √( p̂(1−p̂)(1/n1 + 1/n2) )
where p̂ is the pooled proportion: p̂ = (x1 + x2) / (n1 + n2)
p̂1, p̂2sample proportions for each group
n1, n2sample sizes of each group
x1, x2number of "successes" in each group
Example cue

"Is there a gender difference in seeking mental health treatment?" (outcome is yes/no)

Five-step model
  1. H₀: p1 = p2; HA: p1 ≠ p2
  2. Choose α; find zcritical (±1.96 for α=0.05, two-tailed)
  3. Calculate pooled p̂, then SE
  4. Calculate zobtained
  5. Compare; reject H₀ if |zobtained| > zcritical
Interpretation

"There is [not] a statistically significant difference in [outcome] between [group 1] and [group 2]."

Chi-square test — two categorical variables

χ²

Chi-square test of association

Use when: both variables are categorical and data are presented in a contingency table (counts). Testing whether the two variables are associated.

χ² = Σ ( (O − E)² / E )
E = (row total × column total) / grand total
Oobserved cell count (from your table)
Eexpected cell count (if H₀ were true)
df(rows − 1) × (columns − 1)
Example cue

"Is there an association between having been a victim of robbery and concern about crime?" (both are categorical — yes/no, not/moderate/very concerned)

Five-step model
  1. H₀: the two variables are independent (no association); HA: the two variables are associated
  2. Choose α; calculate df = (r − 1)(c − 1); find χ²critical from chi-square table
  3. Calculate expected counts: E = (row total × column total) / grand total for every cell
  4. Calculate χ²obtained = Σ (O − E)² / E summed over all cells
  5. Compare: reject H₀ if χ²obtained > χ²critical
Interpretation

"There is [not] evidence of an association between [variable 1] and [variable 2] (χ² = [value], df = [value], p [</≥] 0.05)."

Note: Chi-square is always a two-tailed / non-directional test. You cannot specify "greater than" or "less than." The hypotheses are always about independence vs. association.

The five-step model — always use this structure

1
Formulate hypotheses
State H₀ (null) and HA (alternative) using population notation (μ, p, π).
2
Set alpha and find critical value
Choose α (usually 0.05). Use df to look up tcritical or χ²critical, or use z table for zcritical.
3
Calculate the test statistic
Plug numbers into the correct formula to get tobtained, zobtained, or χ²obtained.
4
Make a decision
If the obtained statistic exceeds the critical value (or p < α), reject H₀. Otherwise, fail to reject H₀.
5
State your conclusion
Write a sentence in plain language about what the result means for your research question.
